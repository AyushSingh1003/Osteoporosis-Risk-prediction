# Analysis Module

NHANES-based osteoporosis analysis and model development assets.

## Contents
- `predict-osteoporosis.ipynb`: end-to-end analysis notebook
- `data/`: NHANES source and cleaned datasets
- `images/`: charts used in analysis
- `data_clean_utils.py`, `machine_learning_utils.py`: utility scripts

## Purpose
- Data cleaning and feature engineering
- Model benchmarking and evaluation
- Artifacts used by the Streamlit app
